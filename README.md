# ASI Agent Cloud Skill

Installable skill for ASI Agent Cloud sandbox signup and VM provisioning.

## Install

```bash
npx skills add https://github.com/LloydVickeryASI/asi-agent-cloud-skill --skill asi-agent-cloud
```

## Included

- `asi-agent-cloud/SKILL.md`
- `asi-agent-cloud/agents/openai.yaml`
- `asi-agent-cloud/references/api-quickstart.md`
- `asi-agent-cloud/references/distribution.md`
